Учёт времени идёт смена активна

Смена начата:

0 ч 0 мин 0 сек время в смене
0 часов (для расчёта)
0 ₽ за эту смену

Нет активной смены. Начать смену (вход)

Финансы и расчёт

Ставка: ₽/час

0 рабочих смен
0 всего часов (все смены)
0 ₽ всего начислено

Расчёт: часы × ставка = зарплата (часы округляются вниз до целого).
